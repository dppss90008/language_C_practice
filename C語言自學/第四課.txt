# 流程控制
- 無條件執行一次 (一般述句)
- 有條件執行一次 (if述句)
- 無條件重複執行 (無窮迴圈)
- 有條件重複執行 (while述句)
- 計次重複執行 (for述句)
- 選擇性執行一次 (switch述句)
- 至少執行一次下有條件的重複執行(do-while述句)

# if述句 有條件執行

if(表示式){ 程式片段 }
如果表示式成立 就執行程式片段
